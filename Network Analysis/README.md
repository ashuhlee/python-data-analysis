### Network Analysis
Explores the fundamentals of network analysis using NetworkX and pandas.

- Identify nodes, edges, paths, and cycles in a weighted undirected graph
- Compute weighted degree centrality and ompare nodes based on communication cost
- Calculate betweenness centrality to identify the most-connected bridge node
- Create a network visualization with node sizes scaled according to centrality values
